# Stackhacker UI Portfolio Template

[![Stackhacker UI](https://img.shields.io/badge/Made%20with-Stackhacker%20UI-00DC82?logo=nuxt&labelColor=020420)](https://ui.stackhacker.io)

Use this template to publish a personal portfolio with [Stackhacker UI](https://ui.stackhacker.io), Nuxt, shadcn-vue, and Tailwind CSS v4.

- [Live demo](https://portfolio-template.stackhacker.io/)
- [Documentation](https://ui.stackhacker.io/docs/getting-started)

## Quick Start

```bash [Terminal]
pnpm dlx nuxi@latest init my-portfolio -t gh:stackhacker-ui/portfolio
```

## Setup

Make sure to install the dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.

## Renovate integration

Install [Renovate GitHub app](https://github.com/apps/renovate/installations/select_target) on your repository and you are good to go.
